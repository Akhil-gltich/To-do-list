# 📝 To-Do List Web App

A simple and interactive **To-Do List application** built using **HTML, CSS, and JavaScript**.  
This project helps users manage daily tasks efficiently with features like adding, completing, and deleting tasks.

---

## 🚀 Features
- ➕ Add new tasks  
- ✅ Mark tasks as completed  
- ❌ Delete tasks  
- 🎨 Clean and user-friendly UI  
- ⚡ Instant updates (no reload required)  
- 📱 Responsive layout  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

---

## 📂 Project Structure

To-Do-List/
│── To do list.html
│── README.md


---

## ▶️ How to Run
1. Download or clone this repository  
2. Open **To do list.html** in your browser  

---

## 📸 Overview
This app allows you to:
- Enter tasks in the input field  
- Add them to a list  
- Click on a task to mark it as completed  
- Delete tasks when no longer needed
